### Parameters:

- `"productID"`: `String(int):{}` - productID of the board:
    - `"name"`: `string` - name of the board used for logs.
    - `"baudRate"`: `int` - baud rate for serial communication

### Example config:

``` json
{
  "1234": {
    "name": "Arduino_R4_WIFI",
    "baudRate": 9600
  }
}
```
